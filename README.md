# tic-tac-toe-python
A Python Tkinter-based Tic-Tac-Toe game with GUI, score tracking, and CSV-based game history.

THis consists of player to player tic-tac-toe game.

Locate the main.py file and run the file.
Note: In case you get a _tkinter.TclError, please install pillow using pip install pillow command.

Press the “History” button to get the history of all previous games played. Press the “Exit” button to exit the game. Press the “Start Game” button
